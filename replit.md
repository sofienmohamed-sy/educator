# mathresolver

An AI-powered math tutor that provides step-by-step, curriculum-aware solutions to math problems. Users can upload problems as images, PDFs, or text, select a country-specific curriculum, and receive detailed explanations generated by Anthropic's Claude AI.

## Tech Stack

- **Frontend**: React 18 + Vite + TypeScript, running on port 5000
- **Backend**: Firebase Cloud Functions (Node.js 22 + TypeScript)
- **AI**: Anthropic Claude SDK (multimodal support for images/PDFs)
- **Database**: Cloud Firestore
- **Storage**: Firebase Storage
- **Auth**: Firebase Auth
- **Math Rendering**: KaTeX

## Package Manager

npm workspaces (monorepo)

## Environment Variables

The frontend reads Firebase config from environment variables (prefixed `VITE_`):
- `VITE_FIREBASE_API_KEY`
- `VITE_FIREBASE_AUTH_DOMAIN`
- `VITE_FIREBASE_PROJECT_ID`
- `VITE_FIREBASE_STORAGE_BUCKET`
- `VITE_FIREBASE_MESSAGING_SENDER_ID`
- `VITE_FIREBASE_APP_ID`

In development, the app connects to Firebase emulators automatically (Auth:9099, Firestore:8080, Storage:9199, Functions:5001).

## Development

- Frontend dev server: `npm --workspace apps/web run dev` (port 5000)
- Firebase emulators: `firebase emulators:start`

## Deployment

Configured as a static site deployment:
- Build: `npm --workspace apps/web run build`
- Public dir: `apps/web/dist`
